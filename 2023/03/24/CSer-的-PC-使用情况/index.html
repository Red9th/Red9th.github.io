CSer 的 PC 使用情况

Red9th
2023-03-24
杂

没什么别的，就是换电脑了，重新整理下自己需要使用的各种软件，去掉一些没必要的软件，让电脑环境更加轻松干净。也为了避免下次换电脑的时候配环境又胃疼半天。所以这大约是一个整理软件和配置教程的帖子。

一打开电脑，映入眼帘的只有一个 Edge，总之先把微软账号登了再说。这时候就不得不先谈一下搜索引擎了，而在谈搜索引擎之前，又必须谈到另外一个问题了，那就是魔法。

魔法

是的，在当下这个年代，想要使用各种神奇的玩具，魔法是必不可少的（等一下这是可以说的吗）。

  • Clash for Windows
  • Clash 使用教程

搜索引擎

这方面依然还是选择以 Google 为主，Edge 为辅，毕竟 Google 的插件生态极其丰富，光是这一点就没法让人拒绝。下载好之后登录自己的谷歌账号就行，会同步以前的配置，非常的方便。

  • Google ZN_ch

以及一些推荐的 Google 插件（在 Chrome 应用商店搜索）：

  • Google Translate：可在网页中划词翻译或整句翻译。
  • 新浪微博图床：写博客的时候偶尔会用到，不过最好是建一个自己的图床呢。
  • TeX All the Things：主要用在和 ChatGPT 聊天时，渲染 Latex 公式。
  • Competitive Companion：配合 VScode 中的 CPH 插件，刷题效率蹭蹭涨。
    • 配合 CPH 的食用教程
  • Carrot：Atcoder 的赛时估分软件，打比赛时开一脚有如神助。
  • CF-Predictor：统计自己 CF 的刷题情况。
  • LC Predictor：打完 LeetCode 的比赛之后可以查看自己的 rating 变化。

开发工具

作为一个 CSer，自己的电脑最重要的地方就是用来写代码啦。我按不同的功能来介绍自己的开发工具吧。

版本管理

版本管理应该是后面内容的基础吧，有了这个东西同步以及和别人一起开发什么的也很方便了。

  • Git
  • Git 安装配置教程

然后再下载一个更好使的工具：

  • Github Desktop

用它可以很方便地进行版本管理，而不需要手动输入一堆命令。
JB 家的东西一般都自带了 Git 管理工具，使用起来非常方便。

竞赛向

一个经常打比赛的人需要比较高效的比赛环境，我首推 VScode 进行平时比赛和训练。

  • VScode
  • 环境配置教程

由于设备可以同步，登录账号之后就可以同步之前的配置了，这边大概就不需要进行任何配置了。不过我需要删掉一些不太需要的插件了，例如 Java 相关的。几年下来，Java 系的东西果然还是 JetBrains 做的比较好用。
推荐的一些插件（直接在扩展中搜索下面的名字即可）：

  • Competitive Programming Helper：很好用的刷题插件。
  • Markdown All in One：渲染 MarkDown 用的。
  • Monikai Pro：好看的字体高亮和主题。
  • Git Graph：查看 git 的提交历史。
  • Remote - SSH：ssh 远程连接服务器。

然后就可以把竞赛文件夹的内容同步到 github 上，或者使用 OneDrive 进行同步。

  • DevC++：虽然不常用，但是在某些只能使用这个软件的比赛之前，会用一下找找手感。

Web 开发

扯到前端就不得不先把 Node.js 的环境配置了。

  • Node.js zn_CH

下载后一路“下一步”按默认配置安装即可。安装完成后，cmd 中输入 node -v、npm -v 检查是否安装成功。同时装完这个之后也去把自己的博客捣鼓了下，从旧电脑上把项目搬过来了。

  • 各种语言的 Web 前端开发：WebStorm
  • 小程序/APP 开发（网页包装，比较容易快速出一个成品）：Hbuilder

其他工具：

  • Postman：接口调试。

或者用 VScode 配置 Web 开发环境也行，但是个人还是更喜欢 JB 家的 IDE，版本管理上也非常方便。

Java 开发

总之我这边还是以 Java 8 为主。配置 Java 环境就是下载之后配下环境变量就行了。

  • Java 环境配置教程

然后是下载 IDEA。JB 全家桶可用教育邮箱申请学生认证，每次可免费白嫖一年。

  • IDEA：Java IDE。

要进行 Java 开发的话，Maven/Gradle 之类的项目管理工具也是不可缺少的吧。

  • Maven 配置教程
  • Gradle 配置教程

然后还有数据库的配置啥的。

  • mysql
  • Sqlite
  • navicat

游戏开发

Unity + VScode/VS 2022，正好换电脑了 Unity 也可以下载新版本了。

  • unity download archive
  • Visual Studio 2022

不过最近看麦扣视频，发现 JB 其实也有 .NET IDE，那我必然火速下载了。

  • Rider

科研

啊，想到这个装环境，就有点不想写了，python 和 anaconda 配置教程网上应该一搜一堆吧。

  • Xshell：远程连接服务器。
  • PyCharm：Python IDE。

杂学

Docker：容器，类似于虚拟机的东西。
Fiddler：抓包软件。
live2d：做 2D 模型的软件。
Aegisub：免费字幕软件。

社交

对我来说必不可少的就是 QQ、微信之类的吧，虽然说微信我几乎不用，但是考虑到很多老师喜欢使用微信，还有各种信息交流群之类的，所以还是安装了微信。
QQ 的话稍微配置一下就差不多了，把聊天记录文件夹放在一个比较显眼的、自己容易找到的地方就行了，然后关掉一些没必要显示的东西和功能。微信几乎不太需要配置。

  • QQ
  • 微信

然后就是腾讯会议和钉钉，总是时不时地会用一下。

  • 腾讯会议
  • 钉钉

其他的 Twiter 之类的，通常还是在手机上使用比较舒服。

游戏

作为一个 CSer，打游戏也是生活中必不可少的部分。我平常主要在 Steam 上玩游戏。由于我已经开了魔法，加速器什么的就不太需要了。还有经常听说有人会找错 Steam 的官网欸。

  • Steam zn_CH

都写到这了，那就不得不推荐几款 Steam 游戏了。我最喜欢的类型还是模拟经营和沙盒类游戏：

  • Stardew Valley：最喜欢的模拟经营类游戏！
  • 东方夜雀食堂：很上头，这是我玩的第一款东方游戏，学到了很多东方的知识。
  • Spiritfarer：非常非常温暖的游戏。

然后是两款一直都在玩的沙盒类游戏：

  • 饥荒：很容易沉迷。
  • Terraria：就是好玩！

自己很喜欢的几款 RPG 剧情向冒险游戏：

  • 风来之国 (Eastward)：最喜欢的国产游戏。
  • OPUS：龙脉常歌：很感动的游戏。

其他（其实是不知道该如何划分类别）：

  • Plants vs. Zombies：童年回忆。
  • 双人成行：简单的快乐。
  • Celeste：折磨，但是很享受。

最不擅长的游戏是魂/魂 like 游戏，Hollow Knight 和 Momodora 都在我的库里躺灰好久了。

绘画

  • Aseprite：像素画软件。
  • Sai2：板绘软件。
  • Photoshop：找一个教程安装吧，学生党实在是买不起。
  • PureRef：画图的时候方便找参考图。

杂项

  • 给鼠标指针换个皮肤，换成波奇酱之后就能让人的心情上升好几个点。
  • 打开 Steam 上购买的 Wallpaper Engine，换电脑壁纸会让你的心情更加舒畅。
  • 网易云音乐：写代码的时候听歌心情更愉悦。
  • 百度云网盘：下载各种文件。阿里云盘在这方面好多了，既不限速，又不需要安装客户端，可惜好多资源都是百度云的链接，没办法。
  • OBS Studio：偶尔会需要进行游戏录屏。
  • Simple Sticky Notes：可固定在桌面上的便条，方便平时记录和查看自己的待办。
  • 键盘驱动：取决于自己的键盘。（可以改改键盘预设或者外观啥的）
