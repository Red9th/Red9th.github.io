高斯消元解线性方程组

一般在 $O(n^3)$ 内求得方程组的解。

该方程组的解的情况为：①无解；②无穷多组解；③唯一解。（俺可以回去复习一下线性代数了。。。）

矩阵的初等行变换：

  1. 对调矩阵的 $i, j$ 两行，记为 $r_i↔r_j$
  2. 以不为 $0$ 的数 $k$ 乘以矩阵第 $i$ 行的所有元素，记为 $kr_i$。
  3. 把矩阵第 $j$ 行乘以数 $k$ 加到第 $i$ 行的对应元素上去，记为 $r_i+kr_j$。

经有限次变换可以将原方程组变成上三角形式。

  1. 完美阶梯型：唯一解
  2. 0 = 非零：无解
  3. 0 = 0：无穷多组解

高斯消元求解线性方程组的步骤

枚举每一列的常数：

  1. 找到该列绝对值最大的那一行
  2. 将该行换到最上面
  3. 将该行的这个数变成 1
  4. 将下面所有行的这一列都变成 0
