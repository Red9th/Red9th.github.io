P1969 [NOIP2013 提高组] 积木大赛

这题做法相当的多，其中一种可以用差分来做，这题比较好想到差分上去，但是如何将数组全部变 0 的最小操作数并不是很好想。

首先对于这题来说，其实等同于将该数组全变 0 的最小操作数。若将该数组看成是某个数组的前缀和数组，那么可以求出其差分数组必然是有正有负的，为了让原数组变 0 ，就必须让差分数组全变 0，那就好说了，对正数就减，对负数就加，如果还留下有没有成功变 0 的（即没有配对成功的），那必然就是正负操作数的差值了。所以直接遍历一遍差分数组即可，时间复杂度可以缩到 O(n)。
