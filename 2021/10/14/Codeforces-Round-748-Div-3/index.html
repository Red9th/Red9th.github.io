D1. All are Same（同余）

题意

给定一个数组然后一个正整数 k，求最大的正整数k，每次选一个数减去 k，多次操作之后使得整个数组的数相同，如果这个数字可以任意大就输出 −1。

AC code

这个题完全就是原来做过的一道题，考的都是同余的内容。

要使得：a[1] - n1 * k = a[2] - n2 * k = a[3] - n3 * k = …

即a[1] - a[2] = (n2 - n1 ) * k，a[2] - a[3] = (n3 - n2) * k，…

即a[1] - a[2] 与 a[2] - a[3] 与a[1] - a[3] … 同余，求两两数之间的 gcd 即可。

E. Gardener and Tree（搜索，树论）

题意

有一棵树，每次将其所有的叶子节点删去，k 次后，树上还有多少个节点。

AC code

其实说白了就是看树的深度。每次从度为 1 的所有点开始找，这样从下到上找到每个节点的深度。最后只要是深度大于 k 的就肯定没有被删。我的 ac 代码参考了一个红名大佬的，也学他做了堆优化，用一个队列去遍历，代码看上去清爽了不少，每次遍历记得要改变节点的度数，若又为叶子节点，就继续 push，和 dij 的思想很像，这样就找到了所有点的深度，最后遍历 deep 数组判断一下即可。
