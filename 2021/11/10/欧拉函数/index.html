2021-11-10

包括欧拉函数两种求法和欧拉定理及其推论（费马小定理）。

欧拉函数

$φ(n)$：1 ~ n 中与 n 互质的数。

若 $n=p_1^{α_1}+p_2^{α_2}+…+p_k^{α_k}$，则 $φ(n)=n(1-\frac{1}{p_1})(1-\frac{1}{p_2})…(1-\frac{1}{p_k})$。

根据容斥原理：

1.去掉 $n$ 中所有 $p_1,p_2,…p_k$ 的倍数，即 $n-\frac{n}{p_1}-\frac{n}{p_2}-…-\frac{n}{p_k}$。

2.可能会有数既是 $p_i$ 的倍数，又是 $p_j$ 的倍数，那某些数就会多减掉一次，所以要加上这些个数，即：

$$n-\frac{n}{p_1}-\frac{n}{p_2}-…-\frac{n}{p_k}+\sum_{1≤i,j≤k}\frac{n}{p_ip_j}$$

3.可能会有数是$p_i,p_j,p_k$的倍数，某些数就会多减掉一次，所以也要加上这些个数，即：

$$n-\frac{n}{p_1}-\frac{n}{p_2}-…-\frac{n}{p_k}+\sum_{1≤i,j≤k}\frac{n}{p_ip_j}+\sum_{i≤i,j,k≤k}\frac{n}{p_ip_jp_k}$$

873. 欧拉函数

欧拉筛求欧拉函数

如何在欧拉筛的时候就把欧拉函数求出来呢？

  • 当 i 是质数时，与 i 互质的数有 i - 1 个。
  • 当 i 不是质数时，分两种情况讨论：
    • 若 i 能被枚举到的最小质因子 pj 整除，首先知道 i 的最小质因子是 pj，i * pj 的最小质因子也是 pj，所以求欧拉函数的时候，由于二者质因子完全相同，只是前面的 n 不一样罢了，所以 $φ(i*pj)=φ(i)*pj$。
    • 若 i 不能被枚举到的最小质因子 pj 整除，在前面的质数筛法知识点中，可知 pj 是 i * pj 的最小质因子，而 pj 不是 i 的最小质因子，说明 pj * i 的质因子仅比 i 多了一个 pj，所以 $φ(ipj)=φ(i)(1-\frac{1}{pj})pj=φ(i)(pj-1)$。

时间复杂度：$O(n)$

874. 筛法求欧拉函数

欧拉定理

若 a 与 n 互质，则 $a^{φ(n)}≡1(mod{n})$。

证明：

前置定理：一个数与另外两个数互质，必与这两个数的积互质。

反证：假设 a 和 b 的积 ab 与 c 不互质，那么 ab 与 c 必有公约数 d。
则，c 有一个约数为 d，ab 也有一个约数为 d，可以推断 d 为 a 的约数或 b 的约数，则 a 或 b 和 c 有相同的约数 d；或者 d 为 a 的倍数或 b 的倍数，即 d 的约数为 a 或 b，而 c 有一个约数 d，所以 c 的约数为 a 或 b。

那么 c 和 a 不互质或 c 和 b 不互质，与题目相反，所以假设不成立，则结论成立。

1~n 中和 n 互质的数为：$a_1, a_2, …, a_{φ(n)}$。每个数都乘上 a 就变为：$a a_1, a a_2, …, a a_{φ(n)}$。其中的每一个数均与 n 互质

根据上述定理，$a_1 a_2…a_{φ(n)} mod n=a^{φ(n)}a_1 a_2…a_{φ(n)} mod n=1$，所以二者同余，即：$a_1 a_2…a_{φ(n)}≡a^{φ(n)}a_1 a_2…a_{φ(n)}(mod n)$，化简得 $a^{φ(n)}≡1(mod{n})$，即证。

费马小定理

欧拉定理的推论，当 n 为质数时，$φ(n)=n-1$。

a 为质数，若 a 与 n 互质，则 $a^{n-1}≡1(mod{n})$。
