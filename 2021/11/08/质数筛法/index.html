质数筛法

2021-11-08

朴素筛、埃氏筛、欧拉筛（线性筛）。

朴素筛

筛掉每一个数的倍数。时间复杂度：O(nlogn)，证明如下：
$$\frac{n}{2} + \frac{n}{3} + \frac{n}{5} + … + \frac{n}{n} = n(\frac{1}{2} + \frac{1}{3} + \frac{1}{5} + … + \frac{1}{n}) < n\int_{1}^{n}{\frac{1}{x}}dx = n\ln n < n\log n$$

埃氏筛

筛掉每个质数的倍数。时间复杂度：$O(nloglogn) ≈ O(n)$
由于$1 - n$中有$\frac{n}{\ln n}$个质数，所以时间复杂度近似于将上一种筛法的时间复杂度除$logn$，级别和$O(n)$基本相同。

线性筛

每次对 i 从小到大枚举质因子，那么：
当i % primes[j] == 0时，primes[j]一定是primes[j] * i的最小质因子。
当i % primes[j] != 0时，primes[j]一定小于i的所有质因子，primes[j]也一定是primes[j] * i的最小质因子。
为什么这种筛法是线性的呢？
对于一个合数 x，若 pj 是其最小质因子，那么在枚举 i 时，比起先到 x，一定会先到 x / pj，那么在此时，就会在循环中被x / pj * pj（即被它的最小质因子筛掉）筛掉。所以实际上每个合数只会被其最小质因子筛掉，即每个数只会被筛一次。所以是线性的。
为什么在第二层循环中条件是primes[j] <= n / i呢？
其实是一种防止溢出的写法，实际上表示的是primes[j] * i要小于等于 n 才行，否则里面的 st 数组会越界。
为什么不用在枚举 j 时加上j < idx呢？（换句话说，为什么 j 一定小于 idx 呢？）
因为在枚举到 i 时，小于等于 i 的质数已经全部找到了，而 i 的质因子一定小于等于 i ，所以此时枚举的 j 一定没有超出 idx 的限制，即 primes[j] 是一定存在的。
时间复杂度
若$n ≈ 1e6$，线性筛和埃氏筛的时间效率差不多，若$n ≈ 1e7$，线性筛会比埃氏筛快了大概一倍。
